# AiChatBot
This project is a collection of AI-powered applications built using Large Language Models (LLMs), focusing on real-world use cases like document understanding and interactive chat systems.

The main highlight is a Retrieval-Augmented Generation (RAG) based Book Assistant, which allows users to upload PDF documents and ask questions in natural language. The system retrieves relevant information from the document and generates accurate, context-aware responses, ensuring that answers are grounded in the source content.

In addition to the RAG system, the project also includes a Mood-Based AI Chatbot, where users can interact with an AI that responds in different personalities such as angry, funny, or sad. This demonstrates prompt engineering and conversational AI capabilities.
